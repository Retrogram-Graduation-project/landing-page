Retrogram Device Specifications

Retro devices containing plastic CD cases (also known as jewel cases). They are made of plastic, also known as polystyrene. Lots of common items, including toys, smoke detector cases, kitchenware, medicine bottles. Polystyrene has photorefractive properties.

Features

Retrogram sender application is used to send the Retro device some various assets to be used as hologram such as

2D Images The user can also choose to send a 2D by (2D to 3D image converter)
3D Images The user can send .obj or .sfb files, which are 3D model extensions to present them as holographic materials.
Video The user can send a Video to the Retrogram receiver application to display it.
Text The user can send a Text to the Retrogram receiver application to display it.
Real Time Drawing The user can draw any shape in his device & it appears in the Retro device as a holographic shape.
