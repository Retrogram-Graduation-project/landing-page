More About Retro OS!

Our RetroDevice will work on a Special-Purpose operating system based on the Linux kernel to meet the system’s requirements with the lowest possible operations.

Why develop a custom operating system?
The receiver device should run the receiver application on-startup. The only required features are the Wifi and Bluetooth connection. Removing unnecessary features lowers the consumption and extends battery life. Create custom animation on boot.

The custom operating system is called Retro-OS (Retro operating system). Our Retro-OS is developed using the Android open-source operating system which is based on the Linux kernel. We forked the Android operating system as our current hardware is also a mobile application that is developed to run a mobile operating system.
