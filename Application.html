More About The Applications!

Retrogram application’s main purpose is to control the holographic figure which appears on the Retro device & give more controlling capabilities for the user.

Retrogram Application Consists of two Main Sides:

Sender Application:
Sender Application is used to send to Retrodevice various assets to be used as hologram such as:
2D images
3D images
Videos
Text
Real-time drawing

2D Images:
2D images are "flat", using the horizontal and vertical (X and Y) dimensions, the image has only two dimensions and if turned to the side becomes a line. The user can also choose to send a “2-D to 3-D image converter”, which gives depth Z to the image.

3D Images:
The user can send .obj or .sfb files, which are 3D model extensions to present them as holographic materials.

Text:
The user can send a text to the Retrogram receiver application to display it.

Real-Time Drawing:
The user can draw any shape in his device & it appears in the Retro device as a holographic shape.

Receiver Application:
Aims to process the images or text sent from the Retrogram sender application.

Splitting:
But before processing, the Retrogram receiver application aims to split the media sent into four parts.

Illustration:
The illustration above shows the division of the shape into 4 parts placed at the top, left, right & bottom of the screen.

Enhancement:
Using Image Enhancement techniques, the received media is enhanced and sharpened to appear in a hologram.

Media Control:
The sender device can control the coordinates of the image which means that the user can rotate the media (text, image), zoom in and out, etc...

Connection:
The devices are connected through a LAN connection to share media.
